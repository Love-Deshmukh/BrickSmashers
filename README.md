# 🎮 BrickMasher

BrickMasher is a cross-platform 2D infinite brick breaker game built with Unity.

It features procedural level generation, physics-based gameplay, and modular architecture designed to showcase core game development concepts and clean coding practices.

---

## 🚀 Features

* Infinite procedural level generation
* Physics-based ball mechanics
* Paddle control system
* Modular and scalable architecture
* Cross-platform ready (PC, Mobile, Console)

---

## 🛠️ Tech Stack

* Unity (2D)
* C#
* Visual Studio Code

---

## 🎯 Controls

* PC: A / D or Arrow Keys
* Mobile: Gyroscope (planned)
* Console: Joystick (planned)

---

## 📦 Project Structure

* Assets/ → Game assets & scripts
* ProjectSettings/ → Unity configuration
* Packages/ → Dependencies
